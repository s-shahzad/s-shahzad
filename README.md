## Universal NIDS — Hybrid ML Intrusion Detection

A multi-engine detection system: Suricata + Zeek telemetry feeding a CatBoost classifier, with a signature-anomaly-ML fusion layer. Built for reproducibility — most IDS papers don't replicate; this one does.

| Metric | Result |
|---|---|
| Flows analyzed | 87,432 |
| Runtime | 6 hr continuous soak |
| False positives | 0 |
| Validation replay (509 flows) | 10 alerts · 1.96% alert ratio · 100% reproducible |
| Detection engines | Signature · Supervised ML · Unsupervised ML · Weighted fusion |
| Test coverage | 79% |

Detection content published as Sigma rules, Splunk SPL, Suricata rules, and Zeek scripts — each mapped to the MITRE ATT&CK technique it catches.
